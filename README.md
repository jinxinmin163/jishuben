# 📒 记账本

双生意记账网页应用，适配手机操作。

## 🚲 自行车店记账本 (ike.html)
- 三个分店：东湖印象、太古里、新和县
- 收入/支出/换钱/工资/汇总/分析
- 混合支付（现金+收款码+微信+支付宝）
- 提成按利润计算（售价-成本）× 提成%
- 品类提成比例后台可设置
- 微信推送 + 邮箱通知
- 员工仅能看到2天内数据

## 🍦 蜜雪合伙店记账本 (mixue.html)
- 多币种：KZT / CNY / USD / RMB
- 收入/支出/换汇/股东/工资/汇总/分析
- 股东出资追踪
- 换汇记录
- 订单数/杯数统计

## 🔐 权限
- 管理员密码：dmin888
- 员工无需密码，权限受限

## 🚀 部署
1. 将 *.html 上传到 /var/www/html/
2. 将 mailer.py 放到 /opt/mailer.py，配置 systemd 服务
3. Nginx 配置 /api/mailer 代理到 localhost:8899

## 📡 访问
打开 index.html 选择对应生意入口，或直接访问各页面。
